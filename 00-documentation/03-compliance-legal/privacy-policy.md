# Istabraq Privacy Policy

## 1. Data Collection
We collect:
- Email address (for account creation)
- Name (optional)
- Academic credentials (for scholar verification)
- Uploaded content (books, corrections)

## 2. Data Usage
We use data for:
- Account management
- Scholar verification
- Improving OCR accuracy
- Anonymous usage statistics

## 3. Data Sharing
We DO NOT:
- Sell user data
- Share personal information with third parties
- Use data for advertising

We MAY share:
- Anonymous, aggregated statistics
- Public domain texts with proper attribution

## 4. Copyright Compliance
- We respect all copyright laws
- Rights holders can request takedowns
- Clear attribution for all content
- Public domain marking system

## 5. User Rights
You have the right to:
- Access your data
- Correct inaccurate data
- Delete your account
- Export your contributions

## 6. Contact
For privacy concerns: privacy@istabraq.org
For copyright issues: copyright@istabraq.org
For takedown requests: https://istabraq.org/takedown
